# CMOS Digital Circuits

A collection of fundamental CMOS digital circuit implementations designed using Electric VLSI.

## Implemented Circuits

- CMOS Inverter
- 2-input NAND Gate
- 2-input NOR Gate
- CMOS Latch / Flip-Flop

## Design Tool

- Electric VLSI

## Topics Covered

- CMOS transistor-level design
- Logic gate implementation
- Digital circuit design
- Basic sequential circuits
